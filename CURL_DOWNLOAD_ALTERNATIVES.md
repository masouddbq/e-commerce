# 📥 لینک‌های جایگزین برای دانلود curl

## 🎯 روش 1: دانلود مستقیم از GitHub (پیشنهادی)

### لینک مستقیم دانلود:

**برای Windows 64-bit:**
```
https://github.com/curl/curl/releases/download/curl-8_5_0/curl-8.5.0-win64-mingw.zip
```

یا آخرین نسخه:
```
https://github.com/curl/curl/releases/latest/download/curl-8.5.0-win64-mingw.zip
```

---

## 🎯 روش 2: دانلود از SourceForge

```
https://sourceforge.net/projects/curl/files/
```

1. به این آدرس بروید
2. پوشه `curl-8.5.0` را باز کنید
3. فایل `curl-8.5.0-win64-mingw.zip` را دانلود کنید

---

## 🎯 روش 3: استفاده از WinGet (اگر Windows 10/11 دارید)

```powershell
winget install cURL.cURL
```

---

## 🎯 روش 4: دانلود از سایت‌های معتبر دیگر

### Option A: از سایت اصلی با لینک مستقیم

```
https://curl.se/download/curl-8.5.0.zip
```

### Option B: از Chocolatey (اگر نصب است)

```powershell
choco install curl -y
```

---

## 🎯 روش 5: استفاده از Git (اگر Git نصب دارید)

اگر Git نصب دارید، curl هم همراه آن است:

```powershell
# بررسی Git
git --version

# استفاده از curl از Git
& "C:\Program Files\Git\usr\bin\curl.exe" --version
```

---

## ✅ بعد از دانلود

1. فایل ZIP را استخراج کنید
2. به پوشه `bin` بروید
3. فایل `curl.exe` را کپی کنید
4. در `C:\Windows\System32` قرار دهید
