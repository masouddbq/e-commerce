# سیستم تخفیف‌های ویژه پویا

## توضیحات
این سیستم به جای استفاده از داده‌های ثابت، محصولات تخفیف‌دار را به صورت رندوم از دیتابیس دریافت می‌کند و در سه کارت رنگی نمایش می‌دهد.

## ویژگی‌ها

### 1. دریافت خودکار از دیتابیس
- محصولات با `originalPrice` و `price` متفاوت را پیدا می‌کند
- فقط محصولاتی که واقعاً تخفیف دارند (قیمت فعلی کمتر از قیمت اصلی) نمایش داده می‌شوند
- محصولات به ترتیب تاریخ ایجاد مرتب می‌شوند (جدیدترین اول)

### 2. انتخاب رندوم
- از بین محصولات موجود، سه محصول به صورت رندوم انتخاب می‌شوند
- هر بار که صفحه بارگذاری می‌شود، محصولات متفاوتی نمایش داده می‌شوند

### 3. رنگ‌بندی خودکار
- هر کارت رنگ متفاوتی دارد
- رنگ‌ها از بین 6 طرح رنگی مختلف انتخاب می‌شوند

### 4. مدیریت خطا
- در صورت عدم دسترسی به دیتابیس، از داده‌های پیش‌فرض استفاده می‌کند
- تصاویر محصولات در صورت عدم بارگذاری، به لوگوی برند تغییر می‌کنند

## ساختار دیتابیس مورد نیاز

جدول `products` باید دارای ستون‌های زیر باشد:
- `id`: شناسه یکتا
- `name`: نام محصول
- `price`: قیمت فعلی (تخفیف‌دار)
- `originalPrice`: قیمت اصلی
- `brand`: نام برند
- `image`: تصویر محصول (اختیاری)
- `created_at`: تاریخ ایجاد

## نحوه استفاده

### در کامپوننت SpecialOffers:
```jsx
import { fetchRandomDiscountedProducts } from '../../lib/utils';

useEffect(() => {
  const loadOffers = async () => {
    const discountedProducts = await fetchRandomDiscountedProducts(3);
    setOffers(discountedProducts);
  };
  loadOffers();
}, []);
```

### پارامترهای تابع:
- `limit`: تعداد محصولات مورد نیاز (پیش‌فرض: 3)

### خروجی:
آرایه‌ای از محصولات با ساختار:
```javascript
{
  id: 1,
  title: "نام محصول",
  subtitle: "نام برند",
  discount: "25%",
  originalPrice: "1,000,000",
  newPrice: "750,000",
  image: "/path/to/image.png",
  bgColor: "from-red-500 to-pink-500",
  textColor: "text-white",
  targetLink: "/product/1"
}
```

## تنظیمات

### رنگ‌های پیش‌فرض:
- قرمز به صورتی
- آبی به بنفش
- سبز به فیروزه‌ای
- نارنجی به قرمز
- نیلی به بنفش
- زمردی به فیروزه‌ای

### تصاویر برندها:
در صورت عدم وجود تصویر محصول، از لوگوی برند استفاده می‌شود:
- تویوتا → toyota.png
- هیوندای → hyun.png
- نیسان → nissan.png
- و غیره...

## عیب‌یابی

### مشکلات رایج:
1. **عدم نمایش محصولات**: بررسی کنید که دیتابیس دارای محصولات با `originalPrice` و `price` متفاوت باشد
2. **خطای تصاویر**: فایل‌های تصویر برندها در پوشه `public` موجود باشند
3. **عدم اتصال به دیتابیس**: بررسی کنید که متغیرهای محیطی Supabase تنظیم شده باشند

### لاگ‌های مفید:
- `Loading special offers...`: شروع بارگذاری
- `Loaded discounted products:`: محصولات بارگذاری شده
- `Using fallback offers`: استفاده از داده‌های پیش‌فرض
- `No products with discounts found`: عدم یافتن محصول تخفیف‌دار
